### **Chapter 3: Application Software - Exam Knowledge Points Summary**
**第三章：应用软件 - 考试知识点总结**

---

#### **1. Introduction to Application Software (应用软件简介)**

*   **Application Software (应用软件):** End-user software that is used to accomplish a variety of tasks.
    *   中文解释：指最终用户使用的，用于完成各种任务的软件。
*   **Three Categories of Application Software (应用软件的三大类别):**
    1.  **General-Purpose Applications (通用应用软件):** Widely used programs for common tasks. (例如: 文字处理、电子表格).
    2.  **Specialized Applications (专业应用软件):** Programs that are more narrowly focused on specific professions. (例如: 图形编辑、网页制作).
    3.  **Mobile Apps (移动应用):** Add-on（扩展） programs for mobile devices like smartphones and tablets.

---

#### **2. User Interface (UI) (用户界面)**

The user interface is how you interact with the software.
*   **Graphical User Interface (GUI) (图形用户界面):** Uses graphical elements to interact with the user. Key components include:
    *   **Icons (图标):** Pictorial（图片） representations of commands or programs.
    *   **Pointer (指针):** A symbol on the screen, usually controlled by a mouse.
    *   **Windows (窗口):** Rectangular（矩形的） areas for displaying information and running programs.
    *   **Menus (菜单):** Lists of options or commands.
    *   **Toolbars (工具栏):** A strip of icons used to perform certain functions.
    *   **Buttons (按钮):** Elements you can click to perform an action.
    *   **Dialog Boxes (对话框):** A special window that appears to request information or provide a message.

*   **Ribbon GUI (Ribbon 界面):** A newer interface that replaces traditional menus and toolbars.
    *   **Ribbons (功能区):** Organize commonly used commands into a set of tabs.
    *   **Tabs (选项卡):** Divide the ribbon into major activity areas (e.g., Home, Insert, Page Layout).
    *   **Groups (组):** Each tab is organized into groups that contain related items (e.g., Font group, Paragraph group).
    *   **Contextual Tabs (上下文选项卡):** Tabs that appear only when they are needed for a specific task (e.g., a "Picture Tools" tab appears when you select an image).
    *   **Galleries (库):** Simplify choosing an option by showing the visual effect of applying it.

---

#### **3. Common Features in Applications (应用程序的通用功能)**

*   **Spell Checker (拼写检查器):** Looks for misspelled words.
*   **Alignment (对齐):** Centers, right-aligns, or left-aligns numbers and characters.
*   **Fonts and Font Sizes (字体和字号):** Specify the size and style of text.
*   **Character Effects (字符效果):** Provide a variety of different typefaces and styles (e.g., bold, italics, color).
*   **Edit Options (编辑选项):** Provide easy ways to edit text (e.g., copy, paste, cut).
*   **Find and Replace (查找和替换):** Searches for a specific word or phrase and provides the option to replace it.

---

#### **4. General-Purpose Applications (通用应用软件)**

These are the most common types of application software.

##### **A. Word Processors (文字处理器)**
*   **Definition:** Used to create text-based documents. (中文：用于创建基于文本的文档).
*   **Examples of Documents:** Memos (备忘录), letters (信函), reports (报告), newsletters (时事通讯), brochures (小册子).
*   **Popular Programs:** Microsoft Word, Apple Pages, Google Docs.
*   **Key Features (主要功能):**
    *   **Flyer Features (宣传单功能):** Spell Checker, Center-alignment, Grammar Checker, Fonts, Font Sizes, Character Effects.
    *   **Report Features (报告功能):**
        *   **Captions (图注):** Text that describes a figure or image.
        *   **Footnote (脚注):** A note at the bottom of the page providing more information.
        *   **AutoCorrect (自动更正):** Automatically corrects common typing errors.
        *   **Header or Footer (页眉或页脚):** Text or page numbers that appear at the top or bottom of every page.

##### **B. Spreadsheets (电子表格)**
*   **Definition:** Programs that organize, analyze, and graph numeric data. They create **workbook (工作簿)** files.
*   **Popular Programs:** Microsoft Excel, Apple Numbers, Google Sheets.
*   **Key Features (主要功能):**
    *   **Worksheets (工作表):** The individual pages within a workbook.
    *   **Text Entries (文本条目):** Labels that provide meaning to the values in the worksheet.
    *   **Cells (单元格):** The intersection of a row and column; can contain labels, numbers, formulas, or functions.
    *   **Formulas (公式):** Provide a way to perform calculations (e.g., `=C12-C22`).
    *   **Functions (函数):** Prewritten formulas that simplify complex calculations (e.g., `SUM(C14:C21)`).
    *   **What-if Analysis (模拟分析):** A tool to test the effects of different assumptions in a spreadsheet.
    *   **Chart (图表):** A graphical representation of data in a worksheet.

##### **C. Presentation Graphics (演示图形软件)**
*   **Definition:** Programs that combine visual objects to create attractive, visually interesting presentations. Used to communicate a message and persuade people.
*   **Popular Programs:** Microsoft PowerPoint, Apple Keynote, Google Slides.
*   **Key Features (主要功能):**
    *   **Document Theme (文档主题):** A built-in set of colors, fonts, and effects that can be applied to the entire presentation.
    *   **Templates (模板):** Predesigned styles and layouts to quickly create a presentation.
    *   **Animation (动画):** Adds visual effects to text and objects on a slide.
    *   **Slide Transition (幻灯片切换):** Animates the transition from one slide to the next.

##### **D. Database Management Systems (DBMS) (数据库管理系统)**
*   **Definition:**
    *   **Database (数据库):** A collection of related data.
    *   **DBMS:** A program that structures a database and provides tools to enter, edit, and retrieve data.
*   **Popular Systems:** Microsoft Access, Apple FileMaker, Oracle Database Express Edition.
*   **Key Features (主要功能):**
    *   **Table (表):** The basic structure of a database, with columns (fields) and rows (records).
    *   **Fields (字段):** Columns in a table; they describe the data (e.g., Last Name, First Name).
    *   **Record (记录):** A row in a table; contains all the information about one person or item.
    *   **Primary Key (主键):** A unique identifier for each record in a table (e.g., Employee ID).
    *   **Form (窗体):** An electronic form designed to make it easy to enter and view data for a single record.

#### **5. Specialized Applications (专业应用软件)**

Programs focused on specific professions.

##### **A. Graphics Programs (图形程序)**
*   **Video Editors (视频编辑器):** Used to edit videos to enhance quality and appearance.
    *   **Examples:** Microsoft's Photos, Apple Final Cut Pro, Adobe Premier.
*   **Image Editors (图像编辑器) / Photo Editors (照片编辑器):**
    *   **Definition:** Used for editing or modifying digital photographs.
    *   **Images are made of:** **pixels (像素)**, which form **bitmaps (位图)** or **raster images (光栅图像)**.
    *   **Examples:** Adobe Photoshop, Corel PaintShop Pro.
*   **Illustration Programs (插图程序) / Drawing Programs (绘图程序):**
    *   **Definition:** Create and edit **vector images (矢量图像)**, which use geometric shapes defined by mathematical equations.
    *   **Examples:** Adobe Illustrator, CorelDraw, Inkscape.
*   **Bitmap vs. Vector Graphics (位图 vs. 矢量图):**
    *   **Quality:** Bitmap quality is determined by resolution (分辨率). Vector graphics are resolution-independent.
    *   **Scaling (缩放):** Bitmaps lose quality and become pixelated ("马赛克") when enlarged. Vectors can be scaled infinitely without losing quality.
    *   **Editing (编辑):** Editing a vector object does not affect other objects. Editing a bitmap involves changing individual pixels, which can affect the entire image.
*   **Desktop Publishing Programs (桌面出版程序) / Page Layout Programs (页面布局程序):**
    *   **Definition:** Mix text and graphics to create professional publications like brochures, newsletters, and textbooks. Focuses on page design and layout.
    *   **Examples:** Adobe InDesign, Microsoft Publisher.

##### **B. Video Game Design Software (视频游戏设计软件)**
*   **Purpose:** Helps organize thoughts and guides the user through the game design process, including **character development (角色开发)** and **environmental design (环境设计)**.
*   **Examples:** Unreal Game Engine, Unity, CryEngine SDK.

##### **C. Web Authoring Programs (网页制作程序)**
*   **Definition:** Used for creating a website. Also known as **Web page editors (网页编辑器)** or **HTML editors**.
*   **WYSIWYG:** Stands for "What You See Is What You Get," meaning the editor shows what the final webpage will look like.
*   **Examples:** Adobe Dreamweaver, Microsoft Expression Web.

---

#### **6. Mobile Apps and App Stores (移动应用和应用商店)**

*   **Mobile Applications (Mobile Apps):** Add-on programs for mobile devices like smartphones or tablets.
*   **App Store (应用商店):** A website that provides access to specific mobile apps that can be downloaded.
    *   **Examples:** Apple's App Store, Google Play, Amazon Appstore.

---

#### **7. Software Suites (软件套件)**

*   **Definition:** A collection of separate application programs bundled together and sold as a group. They are usually cheaper than buying the applications individually.
*   **Four Types (四种类型):**
    1.  **Office Suites (办公套件):** Also known as **productivity suites (生产力套件)**. Contain general-purpose applications like a word processor, spreadsheet, etc.
        *   **Examples:** Microsoft Office, Apple iWork.
    2.  **Cloud Suites (云套件):** Also known as **online office suites (在线办公套件)**. Stored on an internet server and can be accessed from anywhere. Good for collaboration.
        *   **Downside:** Dependent on the server and an internet connection.
        *   **Examples:** Google Docs, Microsoft Office 365, Apple iWorks.
    3.  **Specialized Suites (专业套件):** Focus on specific applications, such as graphics suites or financial planning suites.
    4.  **Utility Suites (实用工具套件):** Include programs designed to make computing easier and safer.
        *   **Examples:** Symantec Norton Utilities, AVG Tuneup.
